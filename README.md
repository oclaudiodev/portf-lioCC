Portfólio — Conceitos da Computação

Sobre o Projeto

Este projeto foi desenvolvido como portfólio acadêmico da disciplina Conceitos da Computação.

O objetivo da aplicação é reunir conteúdos, atividades e materiais estudados durante as aulas, utilizando tecnologias modernas de desenvolvimento web.

A aplicação foi construída com foco em:

Organização dos conteúdos Interface moderna e responsiva Aprendizado prático com React Estruturação de projetos Front-End Boas práticas de desenvolvimento

LINK: https://computing-concept.vercel.app
